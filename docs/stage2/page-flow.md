# PalmTag Stage 2 Page Flow

## 页面路由

```text
/
首页

/upload
上传页

/analyzing
分析中页面

/result/[personaId]
结果页

/poster/[personaId]
海报预览页

/donate
打赏页

/privacy
隐私说明页
```

## 用户路径

```text
首页
  ↓
上传图片
  ↓
本地预览
  ↓
点击开始分析
  ↓
分析中页面
  ↓
mock 输出人格
  ↓
结果页
  ↓
海报预览页
  ↓
分享 / 打赏 / 重新测试
```

## 流转说明

首页负责一句话讲清楚 PalmTag，并引导用户进入上传页。

上传页只做本地图片选择和本地预览，不上传服务器。

分析中页面展示 1-2 秒 mock 加载状态，不调用真实千问 API，不伪装成医学检测。

结果页通过 `personaId` 从人格数据读取层获取内容，展示人格名称、金句、最终判断和掌纹特征解释。缺失字段显示 `TODO：需要人工补充`。

海报预览页通过同一个 `personaId` 展示竖版基础海报卡片。Stage 2 只做预览，不做正式 PNG 导出。

打赏页和隐私说明页作为独立路由存在，保证用户可以从结果页或底部入口访问。
