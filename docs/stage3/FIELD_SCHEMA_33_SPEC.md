# Palmmi Stage 3D 33字段工程规范

## 1. 目标

本文只转写 V4.2 的 33 个字段为 Palmmi 工程字段规范，不新增字段、不删除字段、不重命名字段、不重新设计取值范围。

3D 只冻结字段层，不写业务代码，不接 API，不做规则引擎，不做人格匹配，不改 UI。

## 2. 通用规则

- 字段常量名必须完全沿用 V4.2 大写字段名。
- VLM 原始输出层允许字段值为 `null`，后续 3E 负责校验、降级和默认值填充。
- Palmmi 标准化 33 字段层不应保留 `null`。
- 默认降级值按 V4.2 §2.5：二档字段默认 `1`，三档字段默认 `1.5`，四档字段默认 `2`。
- `1.5` 是三档字段降级中位数，不改变原始枚举取值仍为 `0/1/2`。
