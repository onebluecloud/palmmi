# Palmmi Stage 3G 人格规则边界规范

## 1. 目标

本文定义 Stage 3G 的边界。

3G 只定义规则引擎规范，不实现 36 人格详细评分代码。

## 2. 3G 本轮冻结内容

Stage 3G 只冻结：

- 8 母型评分函数。
- 主母型核心字段硬约束。
- 主母型 / 副母型 / 双母型选择契约。
- 母型到人格候选池。
- 12 对相邻人格区分规则。
- 跨母型补判规则。
- 规则引擎输入输出契约。
- 规则引擎错误契约。
- 规则引擎测试计划。

## 3. 3H 后续内容

后续 Stage 3H 才进入：

- 主母型内人格规则实现。
- 跨母型补判细节实现。
- 最终 Top 3 输出结构。
- explainable result 结构。
- 36 人格详细评分代码或工程规则表。

## 4. 禁止事项

3G 不做：

- 业务代码。
- 规则引擎代码。
- VLM 调用。
- 千问 API 接入。
- 端到端识别。
- UI 修改。
- 36 人格详细规则实现。

如果 V4.2 中已有明确代码片段，本轮只转写成规范，不创建代码文件。
