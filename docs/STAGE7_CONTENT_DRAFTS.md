Date: 2026-05-31

Status: `DRAFT_NOT_FOR_PUBLIC_RELEASE`

These are internal drafts for later review. They are not approved publication copy. Do not publish them before Stage 6H real-device acceptance, Stage 6I release-candidate closeout, and user review.

## 1. Voice Rules

Use:

- Light, playful, personal, and low-pressure wording.
- "娱乐向", "人格卡", "海报", "风格标签".
- Clear privacy and limitation notes.

Do not use:

- Fortune-telling claims.
- Medical, relationship, wealth, career, or destiny claims.
- "准确", "科学诊断", "预测", "改命".
- Fear-based copy.

## 8. FAQ Draft

| Question | Draft Answer |
|---|---|
| Palmmi 是手相吗？ | 不是。Palmmi 是娱乐向掌纹人格卡，不做命运预测。 |
| 会预测感情、财运、健康吗？ | 不会。Palmmi 不提供感情、财运、健康、职业等现实决策建议。 |
| 上传照片会保存吗？ | 当前设计不长期保存原图。上传只用于生成本次结果；不要上传包含他人隐私的图片。 |
| 为什么提示照片不清楚？ | 可能是光线暗、掌纹太小、模糊、遮挡或不是掌心正面。换亮一点的环境重新拍。 |
| 为什么提示不是手掌？ | 系统没有识别到清晰的掌心和掌纹。请使用单只手掌、掌心朝向镜头的照片。 |
| 为什么分析超时？ | 网络、图片体积或服务响应可能导致超时。稍后重试，不要连续快速点击。 |
| 结果准吗？ | 它是娱乐人格标签，不是专业判断，也不代表真实人格结论。 |
| 可以分享海报吗？ | 内测阶段只建议小范围分享，不要公开大规模传播。 |
