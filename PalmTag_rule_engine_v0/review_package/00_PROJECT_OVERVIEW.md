# PalmTag 项目整体说明

## PalmTag 是什么

PalmTag 是一个基于真实手掌图片生成“掌纹人格标签”的娱乐型人格产品原型。它不做传统手相的命运、婚姻、健康、财运、寿命判断，而是把可观察的掌纹形态转成结构化字段，再通过规则引擎输出 36 种人格标签之一，并调用对应的展示文案。

## 当前产品定位

当前定位是“轻娱乐 + 人格标签 + 社交分享”的掌纹人格测试。产品重点不是玄学预测，而是让用户通过手掌纹理得到一个有命中感、可讨论、可分享的人格结果。

## 当前技术路线

当前链路为：

```text
真实手掌图片
-> qwen3.6-plus 视觉模型识别 33 字段
-> 同图三次识别投票融合
-> 8 母型评分
-> 36 人格匹配
-> 易混人格分流
-> persona_id 调用展示文案
-> 输出人工评审包/结果报告
```

## 当前阶段进度

项目已完成从 Excel 到规则引擎、从真实手掌图片到人格结果的技术闭环。当前重点已经从“能不能跑通”进入“结果是否有命中感、边界是否稳定”的人工评审阶段。

## 当前默认视觉模型

当前默认视觉模型为：

```text
qwen3.6-plus
```

选择原因：33 字段完整率 100%，与人工基准平均偏差 0.22，M1 集中度较低，P06 不再集中，整体比 qwen3-vl-plus 和 qwen3.5-flash 更适合当前任务。
